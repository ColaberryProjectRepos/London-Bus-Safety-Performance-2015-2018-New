Creating the Calendar Table and establishing relationships between TFL Bus Safety and Calendar Table 14


  1. Click on Modeling from the top Menu bar.  Click on New table. Thae DAX pane will open. Use the following code to create the CalendarTable.
  2. CalendarTable = ADDCOLUMNS(
        CALENDAR(DATE(2015, 1, 1), DATE(2018, 12, 31)),
        "Year", YEAR([Date]),
        "Month", FORMAT([Date], "MMMM"),
        "Month Number", MONTH([Date]),
        "Quarter", "Q" & FORMAT([Date], "Q"),
        "Weekday", FORMAT([Date], "dddd"),
        "Weekday Number", WEEKDAY([Date], 1),
        "Week Number", WEEKNUM([Date], 1),
        "Years", FORMAT([Date], "YYYY")
    )
  3. Click on Date column and change the data type to Date.  change the for Format to short date.
  4. Click on Model view from the left side pane. Establish the many to one relation ship by creating a connect`      between the “Date Of Incident” of TFL Bus Safety v.2 table and the “Date” from CalendarTable.

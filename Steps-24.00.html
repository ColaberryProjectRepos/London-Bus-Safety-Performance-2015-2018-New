 Shape map– Incidents in Borough (Shape map)




  1. Click on Visualizations > Build Visualizations >Shape map > Values>Drag “Borough” from TFL Bus Safety v.2 to Location > Drag “Current year incidents” from TFL Bus Safety v.2 to Color saturation > Format your Visual > Visual > Map settings > Map Type Custom Map.
  2. To add it to the Map. Download the London Borough map from the following link.
  3. https://cartographyvectors.com/map/1179-london-boroughs
  4.  Open Map Shaper, Clock on the link and import the downloaded file. > Export to Geojson type and save to your machine. It will be saved as a json file> Click on Add map > Browse the json file from machine >Fill color >Gradient On and set up as shown below. Color code for Minimum # #5FB9A9, Center #F5C869 and Maximum # #DD6B7F. 
  5. General > Title Off > Effect Transparency 95% > Border On, Color #CCCCCC

Adding Chart – YoY Values (Waterfall Chart) 23

YoY Values (Waterfall Chart)




  1. Create a Table by using the following DAX.
  2. Create a Categories Table.
  3. YoY Categories = DATATABLE(
        "Category", STRING,
        "SortOrder",INTEGER,
        {
            {"Incidents YoY",1},
            {"Fatal YoY",2},
            {"Injuries YoY",3},
            {"Hospitalization YoY",4}
        }
    )
     
    Create a Measure “YoY Values” in the Categories Table.
     
    YoY Values = SWITCH (
        SELECTEDVALUE ('YoY Categories'[Category]),
        "Incidents YoY", 'TFL Bus Safety v.2'[Incidents YoY%],
        "Fatal YoY", 'TFL Bus Safety v.2'[.Fatal YoY%],
        "Injuries YoY", 'TFL Bus Safety v.2'[.Injury YoY%],
        "Hospitalization YoY",'TFL Bus Safety v.2'[.Hospitalization YoY%],
        0
    ) 
  4. Click on Visualizations > Build Visualizations >Waterfall chart > Category>Drag “Category” from YoY categories Table to  Category > Drag “ YoY Values" to Y-axis.
  5. Formatting the chart. Click on the top right three dots of the chart > Unselect the categories and YoY values > Click on the Column “Category” in the Categories table > Click on Column tools on the top menu bar > Sort by column > SortOrder. As an outcome, in the chart the YoY values are arranged in the order of Incident YoY, Fatal YoY, Injuries YoY, Hospitalization YoY.
